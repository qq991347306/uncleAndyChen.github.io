如题

以下，以当前最新的 jdk 版本 8u144 为例。

  1. 进入常规的 java SE 下载页面
  2. 点击页面上的【Technologies】
  3. 点击【Jave SE Platform at a Glance】下面的图片，进入页面：Java Platform Standard Edition 8 Documentation
  4. [然后点击【Downloads】下的【JDK 8 Documentation】] ，进入到Java SE Development Kit 8 Documentation 的下载页面
  5. 点击【Accept License Agreement】前面的单选钮，即可下载。

下载到的文档，在 IDEA 或 Eclipse 下，可以加载，在遇到相应的 java 方法时，按 F1，即可进入对应的文档页面。
