如题

协作开发步骤

  1. 安装 git工具
    • 点击进入 Git for Windows 官网下载最新版
    • 点击进入 TortoiseGit for Windows 官网下载最新版
  2. 进入项目主页，如下所示，复制项目 git 地址。
  3. 进入项目工作目录，拉取（克隆）项目文件。
  4. 直接点 ok
  5. 如果出错，点【retry】，出现 gitee 登录框。
  6. 正常的话，显示 Success，表示 clone 成功
  7. 用 WebStorm 打开项目根目录
  8. 设置项目编码格式，全部为 utf-8
  9. 添加新文件。
    • 第一种方式，文件夹操作。
    • 第二种方式，用 WebStorm 操作。
  10. 提交更新，提交之前需要写备注，最好能体现当前修改的内容，如果无关紧要，仅仅是完善性的修改，建议填：improve
  11. 提交之后，需要推送变更至服务器，只有操作了这一步，其他同事才能获取到你提交的内容。
  12. 获取其他同事提交的变更。建议在修改之前先获取变更，否则，如果在提交时，你提交的文件已经有人更新过了，需要合并，在合并的时候如果有冲突，需要仔细核对。
