使用 Postman 以json形式往后台传递对象，后台接收不到数据

  • Postman download page

  • postman 以 post 请求，将 Headers 的参数 KEY -> Content-Type 的 VALUE 由 application/x-www-form-urlencoded 改为 application/json，如果需要指定编码为 utf-8，改成：application/json; charset=utf-8

  • 后台 Controller 的方法参数前面加上 @RequestBody 就可以了。

但是正常请求会出错：Uncaught TypeError: $.alert is not a function
